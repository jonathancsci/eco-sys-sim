# EcoSysSim

## Set up the dev environment
1. Create a venv named 'venv'
```bash
python3 -m venv venv
```
2. Activate the venv
```bash
source venv/bin/activate
```
3. Install requirements.txt
```bash
pip install -r requirements.txt
```
